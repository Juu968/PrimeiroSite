VisualStudioCode

Visual Studio Code é um editor de código simplificado com suporte para operações de desenvolvimento como depuração, execução de tarefas, e controlo de versões. O seu objectivo é fornecer apenas as ferramentas de que um programador necessita para um ciclo rápido de depuração de código e deixa fluxos de trabalho mais complexos para IDEs mais completos, tais como Visual Studio IDE.
